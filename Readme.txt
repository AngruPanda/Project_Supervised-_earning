Задача 
Разработать модель, которая позволит выявлять снижение покупательной способности
и сделает персонализированные предложения постоянным клиентам, чтобы увеличить их покупательскую активность.

Решение
Для такой задачи классификации возьмем несколько моделей: KNeighborsClassifier(), DecisionTreeClassifier(), LogisticRegression() и  SVC().
С помощью пайплайна подберем оптимальные гиперпараметры и выберем наилучшую модель.
Проанализируем полученные результаты с помощью SHAP, чтобы выявить наиболее выжные для модели признаки и дать рекомендации заказчику

Результат
Kучшая модельпо метрике auc_roc: LogisticRegression c C=1
Наибольшие влияние на модель оказывают следующие признаки:
время на сайте, количество просмотренных страниц и категорий, а также акционные покупки
Был выделен сегмент пользователей приносящих хороший доход и большой вероятностью снижения активности,
а потом изучено нескольких пользователей, чтобы понять что влияет на отнесение к классу снижение активности. 
Подавляющее большинство пользователей, у которых наблюдается снижение активности, это люди которые
в основном покупают по акции и проводят мало времени на сайте. Чтобы сохранить эту аудиторию,
придется постоянно делать какие-то акции для привлечения их внимания.
